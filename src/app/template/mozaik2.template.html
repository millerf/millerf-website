Mozaik - Phase 2 is a domotic project composed focused on media content.

It allows indexing media content across multiple assets (villas and yachts) as well as dynamically controlling Home Atomation equipments.

It is composed of 3 components:
  • A Ionic 1 / AngularJS app for user interface
  • A Local Server installed in each client's household
  • A Cloud server to manage the clients' accounts and assets
